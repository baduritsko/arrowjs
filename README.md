# arrowjs
ArrowMap.js (le nom du projet a évolué parce que le nom arrow.js est déjà utilisé par un autre projet)

Projet de suivi des séances de tir à l'arc :
- enregistrement des volées avec totalisation du score et moyenne (done)
- saisie des valeurs d'angle (heure) et estimation d'un facteur de correction de viseur (in progress)
- synchronisaiton cloud (to do)
- management d'équipe pour les clubs (to do, si besoin)

Installation
------------
Pour le moment le système repose sur des fichiers HTML/CSS/JS, il fonctionne localement (sauvegarde des données dans le cache du navigateur).
